# E-Commerce-Todo-Manager
Python Flask web application implementing RESTful CRUD APIs, SQLite database integration, secure authentication, and task tracking dashboard.

A full-stack task management web application built using Python and SQLite.  
The application supports user authentication, RESTful CRUD operations, and an interactive dashboard for tracking tasks.

---

## 🚀 Features

- User Registration & Login Authentication
- Secure Password Hashing
- Create, Read, Update, Delete (CRUD) Tasks
- RESTful API Endpoints
- SQLite Database Integration
- Dashboard with Task Statistics
- Session Management using Flask-Login

---

## 🛠 Tech Stack

- Backend: Python (Flask)
- Database: SQLite
- Frontend: HTML5, CSS3
- ORM: SQLAlchemy
- Authentication: Flask-Login
- Version Control: Git & GitHub
